# TrailTracker

TrailTracker is a lightweight dark-theme front-end prototype for Account Executives to organize outbound account outreach.

## What it does
- Tracks target accounts and primary contacts.
- Shows how many outreach rounds have already been completed.
- Records channel and outcome details.
- Summarizes open, reply, and meeting-rate analytics.
- Highlights top-performing outreach plays.
- Lets you add new target accounts directly in the UI.

## Run locally
```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in a browser.
